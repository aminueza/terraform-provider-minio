# Data Source: minio_iam_user

Retrieves information about a specific IAM user by name.

Retrieves information about a specific IAM user by name. This is useful for checking user status or using existing users in conditional resource creation.

## Example Usage

```terraform
data "minio_iam_user" "example" {
  name = "existing-user"
}

output "user_status" {
  value = data.minio_iam_user.example.status
}

output "user_details" {
  value = {
    name   = data.minio_iam_user.example.id
    status = data.minio_iam_user.example.status
  }
}

resource "minio_iam_user_policy_attachment" "conditional" {
  count = data.minio_iam_user.example.status == "enabled" ? 1 : 0

  user_name   = data.minio_iam_user.example.id
  policy_name = "readonly-policy"
}
```

## Use Cases

- **User Status Check**: Verify if a user is enabled or disabled
- **Conditional Resources**: Create resources based on user existence or status
- **Import Validation**: Check user details before importing
- **Audit Queries**: Retrieve user information for compliance checks
