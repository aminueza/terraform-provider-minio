# minio_s3_bucket_replication_status (Data Source)

Reads replication status and metrics for a bucket including rule count, replicated/pending sizes, and error counts.

## Example Usage

```terraform
data "minio_s3_bucket_replication_status" "source" {
  bucket = "my-replicated-bucket"
}

output "replication_rules" {
  value = data.minio_s3_bucket_replication_status.source.rules
}
```
