# Data Source: minio_s3_object

Reads an object from a MinIO bucket including its content and metadata.

Reads an object from a MinIO bucket. This is useful for reading configuration files, retrieving content, or checking object metadata.

## Example Usage

```terraform
data "minio_s3_object" "document" {
  object_name = "document.txt"
  bucket_name = "documents-bucket"
}
```

## Use Cases

- **Configuration Retrieval**: Read configuration files stored in MinIO buckets
- **Content Access**: Retrieve object content for use in other resources
- **Metadata Inspection**: Check object properties like size, checksums, and storage class
- **Version Management**: Access specific versions of objects in versioned buckets
