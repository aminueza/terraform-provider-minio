# minio_s3_objects (Data Source)

Lists objects in an S3 bucket with optional prefix, delimiter, and max keys filtering.

## Example Usage

```terraform
data "minio_s3_objects" "logs" {
  bucket    = "my-bucket"
  prefix    = "logs/2026/"
  max_keys  = 100
}

output "log_files" {
  value = data.minio_s3_objects.logs.keys
}

# Browse like a filesystem with delimiter
data "minio_s3_objects" "dirs" {
  bucket    = "my-bucket"
  prefix    = ""
  delimiter = "/"
}

output "top_level_dirs" {
  value = data.minio_s3_objects.dirs.common_prefixes
}
```

### Required

- `bucket` (String) Name of the bucket to list objects from.

### Optional

- `delimiter` (String) Character used to group keys. Use '/' to browse like a filesystem.
- `max_keys` (Number) Maximum number of keys to return.
- `prefix` (String) Limits results to object keys that begin with this prefix.
