# minio_audit_kafka (Resource)

Manages a Kafka target for MinIO audit log forwarding. Audit events are published to the specified Kafka topic.

## Example Usage

```terraform
resource "minio_audit_kafka" "compliance" {
  name           = "compliance"
  brokers        = "kafka1:9092,kafka2:9092"
  topic          = "minio-audit"
  sasl_username  = "minio"
  sasl_password  = var.kafka_password
  sasl_mechanism = "plain"
  tls            = true
}
```
