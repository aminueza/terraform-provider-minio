# minio_iam_idp_ldap (Resource)

Manages an LDAP/Active Directory identity provider configuration for MinIO.

MinIO supports a single LDAP/Active Directory configuration. Once set, users who authenticate via LDAP can have MinIO policies attached through `minio_iam_ldap_user_policy_attachment` and `minio_iam_ldap_group_policy_attachment`.

**NOTE:** See the [MinIO LDAP documentation](https://min.io/docs/minio/linux/operations/external-iam/configure-ad-ldap-external-identity-management.html) for prerequisites and supported LDAP server versions.

## Example Usage

```terraform
resource "minio_iam_idp_ldap" "default" {
  server_addr            = "ldap.example.com:636"
  lookup_bind_dn         = "cn=readonly,dc=example,dc=com"
  lookup_bind_password   = var.ldap_bind_password
  user_dn_search_base_dn = "ou=users,dc=example,dc=com"
  user_dn_search_filter  = "(uid=%s)"
  group_search_base_dn   = "ou=groups,dc=example,dc=com"
  group_search_filter    = "(&(objectclass=groupOfNames)(member=%d))"
}
```

## Schema

### Required

- `server_addr` (String) LDAP server address in host:port format (e.g., 'ldap.example.com:389').

### Optional

- `enable` (Boolean) Whether this LDAP configuration is enabled.
- `group_search_base_dn` (String) Base DN for searching group entries (e.g., 'ou=groups,dc=example,dc=com').
- `group_search_filter` (String) LDAP filter for group membership lookup (e.g., '(&(objectclass=groupOfNames)(member=%d))').
- `lookup_bind_dn` (String) Distinguished name used to bind to the LDAP server for user/group lookups (e.g., 'cn=admin,dc=example,dc=com').
- `lookup_bind_password` (String, Sensitive) Password for the lookup bind DN. MinIO does not return this value on read; Terraform keeps the value from your configuration.
- `server_insecure` (Boolean) Allow plain (non-TLS) LDAP connections. Not recommended for production.
- `starttls` (Boolean) Use STARTTLS to upgrade a plain LDAP connection to TLS.
- `tls_skip_verify` (Boolean) Disable TLS certificate verification. Not recommended for production.
- `user_dn_search_base_dn` (String) Base DN for searching user entries (e.g., 'ou=users,dc=example,dc=com').
- `user_dn_search_filter` (String) LDAP filter to locate a user by username (e.g., '(uid=%s)').

### Read-Only

- `id` (String) The ID of this resource.
- `restart_required` (Boolean) Indicates whether a MinIO server restart is required for the configuration to take effect.

## Import

The LDAP identity provider configuration can be imported using the fixed identifier `ldap`:

```shell
terraform import minio_iam_idp_ldap.default ldap
```

Note: `lookup_bind_password` is not returned by the MinIO API and must be set in your configuration after import.
