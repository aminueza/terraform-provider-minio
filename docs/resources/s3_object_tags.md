# minio_s3_object_tags (Resource)



## Example Usage

```terraform
```terraform
resource "minio_s3_object_tags" "config_file" {
  bucket = minio_s3_bucket.data.id
  key    = "config/app.json"

  tags = {
    Type        = "configuration"
    Application = "webapp"
    Version     = "1.0"
  }
}

  bucket = minio_s3_bucket.logs.id

}
