# minio_server_config_storage_class (Resource)

Manages MinIO storage class configuration for erasure coding parity. Controls data protection levels for STANDARD and REDUCED_REDUNDANCY storage classes.

## Example Usage

```terraform
resource "minio_server_config_storage_class" "main" {
  standard = "EC:4"
  rrs      = "EC:2"
}
```

### Optional

- `comment` (String) Optional comment for the storage class configuration.
- `rrs` (String) Parity for REDUCED_REDUNDANCY storage class (e.g., "EC:2").
- `standard` (String) Parity for STANDARD storage class (e.g., "EC:4" for 4 parity drives).
